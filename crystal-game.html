Crystal Collector

    Instruction

  • You will be given a random number at the starting of the game.
  • There are four crystal buttons,by clicking on them you will add a specific number of points to your score
  • You win the game if you match the your score with the random number given at the start.
  • You lose the game if your score goes above the random number given
  • The value of each crystal is hidden from you till until you click it
  • Each time the game starts, the game will change the value of each crystal
